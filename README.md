# GratitudeChain 🔗

An AI-powered gratitude journaling and habit tracking app built for the CWB Hackathon 2026.

## Prior Work Disclosure
The GratitudeChain concept and business plan were developed 
prior to the hackathon. All code in this repository was written 
exclusively during the hackathon period (April 2 – May 3, 2026).
No prior codebase was used.

## Tech Stack
- **Microsoft Foundry** — AI model hosting (gpt-5.4-mini)
- **Microsoft Agent Framework** — 5 specialized AI agents
- **Azure Cosmos DB** — User data and entries storage
- **Azure AI Search** — Semantic memory resurface
- **Azure App Service** — API hosting
- **Azure Functions** — Background jobs
- **FastAPI** — REST API

## 5 AI Agents
| Agent | Role |
|-------|------|
| Reflection Agent | Generates daily gratitude prompts |
| Streak Agent | Tracks habit chain and milestones |
| Memory Agent | Resurfaces past entries on hard days |
| Mindfulness Agent | Guides breathing exercises |
| Insights Agent | Monthly Gratitude Portrait |
